How to build APK:

1. Install Android Studio.
2. Open this folder: creatorlift_admin_android_webview_source
3. Let Gradle sync finish.
4. Go to Build > Build Bundle(s) / APK(s) > Build APK(s).
5. APK will be generated inside app/build/outputs/apk/debug/

Note: This is a WebView APK using the same admin panel files.
